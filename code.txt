import React
Reactをインポートしています。Reactは、ユーザーインターフェースを作成するためのJavaScriptライブラリです。

{ useEffect, useState, useCallback }
Reactの特定のフック（Hooks）をインポートしています。これらのフックは、関数コンポーネントで状態管理や副作用の処理を行うために使用されます。

useEffect
コンポーネントの状態を管理するためのフックです。例えば、ユーザー入力やAPIからのデータを保持するために使用されます。useStateは、状態変数とその更新関数を返します。

useCallback
メモ化されたコールバックを返すためのフックです。特定の依存関係が変更されない限り、同じコールバック関数を再利用します。これにより、無駄な再レンダリングを防ぐことができます。
